# Package index

## All functions

- [`rgennorm()`](http://bsvars.org/StealLikeBayes/dev/reference/rgennorm.md)
  : Samples random draws from a generalised normal distribution using
  the Gibbs sampler by Waggoner & Zha (2003a)
- [`rgig1()`](http://bsvars.org/StealLikeBayes/dev/reference/rgig1.md) :
  Samples random draws from the generalized inverse Gaussian
  distribution
- [`rhaar1()`](http://bsvars.org/StealLikeBayes/dev/reference/rhaar1.md)
  : Sample random matrices from Haar distribution
- [`rmvnorm1_precision_sampler()`](http://bsvars.org/StealLikeBayes/dev/reference/rmvnorm1_precision_sampler.md)
  : Samples random draws from a multivariate normal distribution using
  the precision sampler by Chan & Jeliazkov (2009)
- [`rtmvnorm()`](http://bsvars.org/StealLikeBayes/dev/reference/rtmvnorm.md)
  : Sample Random Draws From the Truncated Multivariate Normal Using the
  Algorithm Proposed by Yifang Li and Sujit K. Ghosh (2015)
- [`rtmvnorm_hmc()`](http://bsvars.org/StealLikeBayes/dev/reference/rtmvnorm_hmc.md)
  : Generate Truncated Multivariate Normal Samples via Hamiltonian Monte
  Carlo
- [`sample_variances_horseshoe()`](http://bsvars.org/StealLikeBayes/dev/reference/sample_variances_horseshoe.md)
  : Samples variances from the horseshoe prior using Gruber & Kastner
  (2024)
- [`sample_variances_normal_gamma()`](http://bsvars.org/StealLikeBayes/dev/reference/sample_variances_normal_gamma.md)
  : Samples variances of the Normal-Gamma prior distribution by Brown &
  Griffin (2010).
